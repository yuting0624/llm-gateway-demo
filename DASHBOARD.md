# Looker Studio ダッシュボード設定ガイド

## BigQuery テーブルとの接続

1. [Looker Studio](https://lookerstudio.google.com/) を開く
2. 「空のレポート」を作成
3. データソースの追加 → BigQuery を選択
4. プロジェクト: `data-agent-bq` → データセット: `llm_gateway` → テーブル: `spend_logs`
5. 「追加」をクリック

## 推奨チャート

### 1. モデル別コスト（円グラフ）
- **ディメンション**: `model`
- **指標**: `SUM(spend)`
- **用途**: Claude vs Gemini のコスト比率を一目で把握

### 2. ユーザー別コスト（横棒グラフ）
- **ディメンション**: `user`
- **指標**: `SUM(spend)`
- **並べ替え**: コスト降順
- **用途**: 誰が一番使っているか

### 3. リクエスト数の時系列（折れ線グラフ）
- **ディメンション**: `startTime`（日付に変換）
- **指標**: `COUNT(request_id)`
- **内訳ディメンション**: `model`
- **用途**: 利用トレンド、モデル別の推移

### 4. レイテンシ分布（ヒストグラム）
- **計算フィールド作成**: `TIMESTAMP_DIFF(endTime, startTime, MILLISECOND)`
- **ディメンション**: 上記の計算フィールド
- **用途**: パフォーマンス監視

### 5. トークン消費量（スコアカード + 時系列）
- **指標**: `SUM(prompt_tokens)`, `SUM(completion_tokens)`, `SUM(total_tokens)`
- **用途**: トークン効率の分析

### 6. チーム別月次コスト（テーブル）
- **ディメンション**: `team_id`, `DATE(startTime)` (月)
- **指標**: `SUM(spend)`, `COUNT(request_id)`, `AVG(total_tokens)`
- **用途**: 部門別のチャージバック

## フィルター設定

以下のフィルターをダッシュボードに追加すると便利：
- **期間セレクタ**: `startTime` ベース
- **モデルフィルター**: `model` のドロップダウン
- **ユーザーフィルター**: `user` のドロップダウン
- **チームフィルター**: `team_id` のドロップダウン
